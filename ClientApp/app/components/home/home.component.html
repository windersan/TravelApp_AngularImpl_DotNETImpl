I used REST via a Web API host and Angular2 client to set up some web service infrastructure, an example is in "AccountController.cs", there I set up some HTTP that is called in "account.component.ts". Everything is hosted on Microsoft Azure.

You can also see the API in action by going to http://main52.azurewebsites.net/api/account/amt/1000 you can replace 1000 with any number you like, just put it in your browser and you should see a JSON object displayed by your browser, this is the same url that the client is using.
